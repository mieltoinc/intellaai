Run shell commands in the sandbox using the provider instance returned by `initializeSandbox`.

## Basic usage

```typescript
import { Intella, SandboxProviderType } from 'intella';

const intella = new Intella();
const sandbox = await intella.initializeSandbox(SandboxProviderType.LOCAL, { cwd: process.cwd() });

const result = await sandbox.executeCommand('npm test', {
  cwd: '/app',
  env: { NODE_ENV: 'test' },
  timeout: 60_000,
});

console.log(result.exitCode, result.stdout, result.stderr, result.executionTime);
await sandbox.close();
```

## Options

| Option | Type | Description |
|--------|------|-------------|
| `cwd` | string | Working directory for the command |
| `env` | Record\<string, string\> | Environment variables |
| `timeout` | number | Timeout in milliseconds |

## Result (CommandResult)

| Field | Type | Description |
|-------|------|-------------|
| `exitCode` | number | Process exit code |
| `stdout` | string | Standard output |
| `stderr` | string | Standard error |
| `executionTime?` | number | Execution time in milliseconds |

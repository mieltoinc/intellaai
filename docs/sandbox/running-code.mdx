---
title: "Running code"
description: "Execute interpreted code (Python, JavaScript) in the sandbox"
---

Run interpreted code (e.g. Python, JavaScript) in the sandbox using the sandbox instance returned by `initializeSandbox`. There is no Intella facade for `runCode`—call it directly on the provider.

<Info>
  All code samples use the `intella` package. Install with `npm install intella` and import from `'intella'`.
</Info>

## Basic usage

```typescript
import { Intella, SandboxProviderType } from 'intella';

const intella = new Intella();
const sandbox = await intella.initializeSandbox(SandboxProviderType.E2B, { templateId: 'base' });

const codeResult = await sandbox.runCode(
  'print(2 + 2)\nimport sys\nprint(sys.version)',
  {
    language: 'python3',
    timeout: 30_000,
    onStdout: (line) => console.log('stdout:', line),
    onStderr: (line) => console.error('stderr:', line),
  }
);

console.log(codeResult.text, codeResult.results, codeResult.stdout, codeResult.stderr);
await sandbox.close();
```

## Options

| Option | Type | Description |
|--------|------|-------------|
| `language` | string | e.g. `'python3'`, `'javascript'` (provider-dependent) |
| `timeout` | number | Timeout in milliseconds |
| `onStdout` | (line: string) => void | Callback for stdout lines |
| `onStderr` | (line: string) => void | Callback for stderr lines |

## Result (CodeExecutionResult)

| Field | Type | Description |
|-------|------|-------------|
| `text?` | string | Main result text |
| `results` | Array<{ text?, html?, markdown? }> | Result items |
| `stdout` | string[] | Stdout lines |
| `stderr` | string[] | Stderr lines |
| `error?` | { name, value, traceback } | Error details if execution failed |
| `executionTime?` | number | Execution time in milliseconds |

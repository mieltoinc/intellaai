Using agents in the **vanilla** way means no sandbox: the agent runs in your process, like calling the OpenAI API or the AI SDK. There is no isolated environment, no command execution, and no file or git operations in a remote sandbox. For sandbox-backed agents, use [SandboxAgent](/sandbox-agent).

<Info>
  All code samples use the `intella` package. Install with `npm install intella` and import from `'intella'`.
</Info>

## Quick start

Create an **Intella()** instance, **select** an agent by type and optional config, then **execute** or **stream** a task.

```typescript
import { Intella, AgentType } from 'intella';

const intella = new Intella();
intella.selectAgent(AgentType.CLAUDE, { model: 'sonnet' });

const response = await intella.executeTask({ prompt: 'Explain quantum computing in 2 sentences.' });
console.log(response.text, response.agentType, response.metadata);
```

## Execute and stream

- **executeTask** — Returns a **TaskResponse** with `text`, `agentType`, and optional `metadata` (e.g. usage, finishReason).
- **streamTask** — Returns an async iterable of chunks; use `chunk.text` and `chunk.isDone` to stream the response.

```typescript
for await (const chunk of intella.streamTask({ prompt: 'Write a short poem.' })) {
  if (!chunk.isDone) process.stdout.write(chunk.text);
}
```

**TaskRequest** can include `prompt`, `systemPrompt?`, `messages?`, `temperature?`, `maxTokens?`.

## Agents overview

| Agent | AgentType | Default model | Auth | Page |
|-------|-----------|----------------|------|------|
| **Claude Code** | `AgentType.CLAUDE` | sonnet | `claude login` CLI | [Claude Code](/agents/claude-code) |
| **Codex** | `AgentType.CODEX` | gpt-5.2-codex | OPENAI_API_KEY | [Codex](/agents/codex) |
| **OpenCode** | `AgentType.OPENCODE` | anthropic/claude-sonnet-4-5 | `opencode login` or provider keys | [OpenCode](/agents/opencode) |

All are used the same way: `intella.selectAgent(AgentType.*, config)` then `intella.executeTask(request)` or `intella.streamTask(request)`.

## Optional: orchestration

Run multiple agents in sequence or parallel with **orchestrateTask**:

```typescript
const result = await intella.orchestrateTask(
  { prompt: 'Analyze this code' },
  [AgentType.CLAUDE, AgentType.CODEX],
  'parallel',
  { combineStrategy: 'merge' }
);
```
